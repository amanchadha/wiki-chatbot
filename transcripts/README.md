# Claude Code transcripts

Human-readable HTML export of the Claude Code session used to build this
project. Open `index.html` in a browser; `page-001.html` through
`page-013.html` are the session in chronological order.

The session covers: v0 plan + build, eval-harness design + judge prompt
review, the baseline run and iterations v1 through v1.3 (including the
multi-hop extension), the judge-v4/v5 hallucination + citation grading
work, citation mode with claim-level linked citations, the web UIs and
Vercel deployment, and the docs/packaging pass. Every eval run referenced
in the transcript has its artifacts checked in under `eval/runs/`.
